# primus-network-js-sdk-dapp skill

Scaffold a Primus Network zkTLS DApp that proves off-chain data from a template ID.

## Install

```
Set up the primus-network-js-sdk-dapp skill from https://github.com/primus-labs/skills.git, install its dependencies, and verify it with the doctor script.
```

Or manually:

```bash
git clone https://github.com/primus-labs/skills.git
cd skills/primus-network-js-sdk-dapp
npm run doctor
```

## Use

```
Use the primus-network-js-sdk-dapp skill to scaffold a DApp that proves <what_to_prove>. The template ID is <template_id>.
```

Example:

```
Use the primus-network-js-sdk-dapp skill to scaffold a DApp that proves Binance account ownership. The template ID is e790545d-eda1-4569-863f-2a6d280b5386.
```

## Verify

```bash
npm run doctor
```

All checks must pass before using the skill.
